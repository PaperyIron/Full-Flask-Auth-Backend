# Flask BackEnd Authorization Lab

## To Run
1. Clone repository to your local machine.
2. Open a terminal in FULL-FLASK-AUTH-BACKEND
3. Run 'pipenv install'
4. Run 'pipenv shell'
5. cd into the client-with-sessions folder.
6. Run 'npm install'
7. Run 'npm start'
8. Open a new terminal window in FULL-FLASK-AUTH-BACKEND
9. cd into the server folder
10. Run 'python app.py'
11. Open 'http://localhost:4000' in your browser to view the website.
